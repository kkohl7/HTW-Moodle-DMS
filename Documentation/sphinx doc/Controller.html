Controller module¶

class Controller.Controller(path: str)¶

Bases: object

The Controller class is for loading the specific guis from the GUI folder. It also implements and fill the required GUI with logic.

cleanMainWindow()¶

reset all fields

errorMessage(e: str)¶

open the error Gui and print the error message

Parameters:e – the error message
goBackMainPage()¶

goBackMainPage will start the MainPage and will connect everything with the required functions and print the required information to the gui.

openAllData()¶

open all Data which are in the tab_data(modle = modelCourseData) printed If no data is printed inform the user that he has to search for CourseData to open a data

openCurrentDownload()¶

Will open the current download gui and connect the scrapy logger to the gui. It start also the Crawler, if the user stored all required information.

openData()¶

open the the selected CourseData if None is selected inform the user that he has to select a CourseData in the table

openHelp()¶

open the help GUI. This is for helping the user to work with the application

openMoodleSummary(destination: str = 'Main Page')¶

Open the GUI for the getMoodleSummary. And print all informations to the gui. This GUI is for informing the user about what data per course is new, old and could not be downloaded. Also every specific error in the download process will be shown.

Parameters:destination – Which GUI shall load after this gui.
openSettings()¶

open the GUI for settings. It is for saving the user information password and username from moodle.

openTextProcessing()¶

open the GUI CurrentDownload and start the text processing. Also it connects the logger to the edit line. This Gui informs the user about which CourseData will be textually processed.

openWindowStartDownload()¶

open the gui to ask the user, if he really wants to download the moodle data. It will connect everything with the required functions and print the required information to the gui.

printCourseData(table='')¶

print all CourseData from the table to the table tab_data.

Parameters:table – the table which shall be printed
printGeneralCourseInformation()¶

print the generalCourseInformation to the table tab_courses

quitProgram()¶

Tries to delete every file in the temp folder. Only the files, which are not any more in use will be deleted. After this quit the program.

saveAllData()¶

save all CourseData which are printed in the tab_data(modle = modelCourseData) If no data is printed inform the user that he has to search for CourseData to open a data

saveData()¶

save the selected data to a folder, which the user select. if None is selected inform the user that he has to select a CourseData in the table

saveSettings()¶

It tries to save the informations from the settings to the database. If every field is filled save the information to the database and go back to the main page. Otherwise print an error message and do nothing.

searchData(filter: str)¶

search for data if filter is all search for all data for search string otherwise for the selected filter after this print the searched data to table tab_data

Parameters:filter – filter = “All” if all data shall be searched; filter = “Cell” if only for the specific

selected filter data shall be searched

selectCourseData(signal)¶

set the selectedCourseData on the data which was selected. Print the abstract, the frequency of words and the errors from the data to the text field “text_Abstract”

selectGeneralCourseInformation(signal)¶

Set the selectedFilter on the selected Value, which the user selected. And print all Data for the specific filter to the tab_data

showDataWithoutTextMining(filter: str)¶

Search for data without text mining and print them to the table tab_data.

Parameters:filter – filter = “All” if all data without text-mining shall be searched; filter = “Cell” if only for the specific

selected filter data without text-mining shall be searched
