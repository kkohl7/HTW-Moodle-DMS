DB.IDatabaseManager module¶

class DB.IDatabaseManager.IDatabaseManager¶

Bases: object

The IDatabaseManager is for the communication with the deposited database The functions are:

  • create all Tables for the model
  • check if a object is already stored in the database
  • insert object in the database
  • get objects from the database
  • delete objects from the database
