frequency_Words(text: str, n: int, language: str) → str¶

frequency_Words compute the frequency of each word in text and give the n words with the highest pop up back as a string.

Parameters:
  • text – The text for which the n highest words shall be extract
  • n – number of words which shall be extract
  • language – The language of the text
Returns:

Returns a String, where each row contains the word and how often it occurs
Raise:

ExceptionTextAbstraction: When n < number of words in text
